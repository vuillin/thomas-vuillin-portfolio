Site événementiel - Quantic Grand Prix

VueJS NodeJS JavaScript
Page d'accueil du site.
Carte interactive de l'événement.
Page d'information.
Backoffice et gestion des services.

Ce projet universitaire est un site web basé sur un événement fictif de Formule 1 nommé Quantic Grand Prix, inspiré des événements existants mais enrichi de nouvelles activités inédites.

Il s'agit d'une application web full-stack moderne, avec un front-end en Vue.js conçu comme une Single Page Application. Cette interface communique avec une API REST en Node.js/Express, qui centralise la logique métier et assure l’échange de données avec une base MySQL.

L’application inclut un système de rôles permettant de gérer différents niveaux d’accès selon les profils utilisateurs. La base de données repose sur une structure pensée pour supporter l’évolution de l’événement, permettant d’ajouter de nouvelles fonctionnalités ou activités sans refonte majeure. La sécurité occupe aussi une place essentielle : authentification, autorisations, protection des données sensibles et respect des bonnes pratiques de développement.

Color Harmony Generator

Python
Générer des palettes grâce à une couleur.
Générer des palettes grâce à une image.

Color Harmony Generator est une application python conçue pour aider les designers et développeurs à créer instantanément des palettes de couleurs esthétiques et cohérentes. Elle simplifie le choix chromatique grâce à la précision d'algorithmes mathématiques et la puissance de l'intelligence artificielle pour garantir un résultat harmonieux.

L'outil propose trois fonctionnalités majeures : la génération d'harmonies classiques (complémentaires, pastel, etc.) depuis un code couleur, l'extraction intelligente des teintes dominantes d'une image via le Machine Learning, et un mode de création aléatoire. Il permet une visualisation des résultats en temps réel et l'exportation automatique des palettes générées sous forme de planches haute définition.
